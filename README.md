# Grimoire Stock

Application web React/Vite pour la gestion des commandes et stocks de laboratoire.

## Fonctionnalites

- Commande de consommables et reactifs par categorie
- Ajout d'articles libres
- Calcul automatique des montants et du total general
- Affichage en FCFA avec separateur de milliers par points
- PDF de commande avec informations laboratoire et expedition
- Historique local des commandes
- Catalogue editable en JSON
- Themes visuels
- Interface mobile-friendly et PWA de base

## Developpement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploiement Vercel

Le projet est un frontend Vite statique. Le build de production est genere dans `dist/`.
